# Tecnologia utilizada

- React


## Implantação em produção
- Serviço: Netlify 

# Como executar o projeto

Pré-requisitos: 
- React 18.1.0

```bash
# clonar repositório
git clone git@github.com:Adyson-Lima-Programador/filmes.git

# entrar na pasta filmes
cd filmes

# executar o projeto
npm start
```
